# drorbrook.com

Personal website of Dror Brook — engineering manager, podcaster, writer, and mentor.

## Sections

- **Home** — intro and links out to everything
- **About** — bio and background
- **Writing** — curated links to posts published on Medium, giffgaff, Monzo, etc.
- **Podcast** — Beyond the Sprint
- **Talks** — video lectures (Tech Gym, etc.)
- **Mentoring** — book a mentoring session via an external scheduler
- **Contact** — LinkedIn & GitHub

## Tech stack

- Next.js (App Router) + TypeScript
- Tailwind CSS
- Deployed on Vercel

## Development

```bash
npm run dev     # local dev server
npm test        # run test suite
npm run build   # production build
```
